Licenses

This page provides license information for this app and its third-party dependencies.

  • Project license: PROJECT_LICENSE.txt
  • External runtime assets: EXTERNAL_ASSETS.md
  • Third-party licenses (runtime / production): third-party-licenses.prod.json
  • Third-party licenses (dev-only): third-party-licenses.dev.json
  • Third-party licenses (all installed): third-party-licenses.all.json

The third-party license lists are generated from the installed dependency tree using pnpm licenses list --json and sanitized to avoid embedding local filesystem paths.
